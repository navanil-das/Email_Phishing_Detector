# Phishing Tester

A modern AI-powered phishing detection web application that helps users identify suspicious emails and URLs using machine learning models and real-time analysis.

## Live Demo

**Website:** [https://phishingtester.vercel.app/](https://phishingtester.vercel.app/)

---

# Features

* Detect phishing emails instantly
* Analyze suspicious URLs
* Machine learning powered prediction system
* Clean and responsive UI
* Real-time API-based analysis
* Confidence score for predictions
* User-friendly interface for testing emails and links
* Fast and lightweight frontend

# How It Works

The application uses a trained machine learning model to classify whether an email or URL is:

* Legitimate
* Suspicious

Users can paste:

* Email sender information
* URLs or links

The system extracts features from the input and sends them to the prediction API, which returns the phishing probability and classification.

# Installation

## Clone the Repository

```bash
git clone https://github.com/navanil-das/Email_Phishing_Detector.git
cd Email_Phishing_Detector
```

## Install Frontend Dependencies

```bash
npm install
```

## Run Frontend

```bash
npm run dev
```

---

# Backend Setup

Install backend dependencies:

```bash
npm install
```

Run backend server:

```bash
npm start
```

---

# ML Model Setup

Install Python dependencies:

```bash
pip install -r requirements.txt
```

Run the ML server:

```bash
python app.py
```
